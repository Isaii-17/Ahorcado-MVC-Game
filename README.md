#Lógica de Programación UIDE
**Nombre del Proyecto:** Juego del Ahorcado Utilizando Visual Studio Code  
**Nombre del Estudiante:** Steven Isai Caina Yanzapanta  
**Objetivo:** Desarrollar un juego interactivo "Juego del Ahorcado" en entorno de consola utilizando Python para aplicar de forma práctica los fundamentos de la programación orientada a objetos y la utilización estructurada de flujos de datos.  
**Descripción:** En este proyecto implementamos un sistema jugable por turnos donde se programó un banco de palabras técnicas, un control preventivo para bloquear caracteres inválidos o repetidos y un renderizado gráfico dinámico según los fallos del usuario, todo esto desarrollado de forma modular aplicando el patrón arquitectónico Modelo-Vista-Controlador (MVC) para asegurar que la lógica de las reglas funcione de manera independiente a la interfaz visual de la terminal.
